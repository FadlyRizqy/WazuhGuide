Panduan Instalasi & Konfigurasi Wazuh

SIEM + FIM + Integrasi Teler, Suricata & Telegram

Pendahuluan

Wazuh adalah platform keamanan open-source untuk mendeteksi dan merespons ancaman, menganalisis log, serta mengintegrasikan berbagai tool keamanan lainnya. Panduan ini disusun untuk membantu siapa saja membangun sistem keamanan berbasis Wazuh yang kuat.

Instalasi Wazuh Server & Agent

# Server
curl -sO https://packages.wazuh.com/4.7/wazuh-install.sh && sudo bash ./wazuh-install.sh -a

# Agent Ubuntu
curl -s https://packages.wazuh.com/key/GPG-KEY-WAZUH | gpg --no-default-keyring --keyring gnupg-ring:/usr/share/keyrings/wazuh.gpg --import
echo "deb [signed-by=/usr/share/keyrings/wazuh.gpg] https://packages.wazuh.com/4.x/apt/ stable main" | tee -a /etc/apt/sources.list.d/wazuh.list
apt-get update
WAZUH_MANAGER="IP_ADDRESS" apt-get install wazuh-agent
systemctl start wazuh-agent

File Integrity Monitoring (FIM)

FIM digunakan untuk mendeteksi perubahan tak sah pada file atau folder sistem.

<directories realtime="yes" check_all="yes" report_changes="yes">/var/www/html</directories>

Integrasi dengan Teler

Teler adalah IDS HTTP ringan. Digunakan bersama Wazuh untuk mendeteksi serangan web berbasis signature.

# Unduh dan konfigurasi
wget https://github.com/kitabisa/teler/releases/download/...tar.gz
tar -xvzf ...
mv teler.example.yaml teler.yaml
tail -f /var/log/apache2/access.log | ./teler -c teler.yaml

Integrasi dengan Suricata

Suricata memberikan deteksi dan pencegahan serangan jaringan (NIDS/IPS). Konfigurasi log JSON dan integrasikan ke Wazuh:

<localfile>
  <log_format>json</log_format>
  <location>/var/log/suricata/eve.json</location>
</localfile>

Integrasi dengan Telegram

Gunakan bot Telegram untuk menerima alert secara langsung:

  • Buat bot via @BotFather
  • Dapatkan Group ID dari @myidbot
  • Tambahkan skrip Python untuk kirim pesan

Simulasi Serangan

  • Webshell: Upload skrip reverse shell ke webserver dan pantau dari netcat.
  • Nmap Scan: Jalankan nmap -sS --script=vuln ke endpoint Wazuh Agent.
  • Web Attack: Jalankan nikto untuk simulasi serangan aplikasi web.
